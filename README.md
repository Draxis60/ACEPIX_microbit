## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/draxis60/ACEPIX_microbit** and import

## Edit this project

To edit this repository in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/draxis60/ACEPIX_microbit** and click import
 
 ## Instructions
* a moves on the x-axis and b moves on the y-axis
* Touching the logo places a pixel
* Holding the logo and pressing A and B together clears the canvas
* pressing the pins changes a pixel's brightness
